# credit-card-fraud-detection
# Project Overview
This project is a Machine Learning based web application that detects fraudulent credit card transactions.The system allows users to input transaction details and predicts whether the transaction is Fraud or Not Fraud using a trained ML model.The application is deployed using Streamlit Cloud.

# Problem Statement
Credit card fraud is a major issue in the financial sector. Fraudulent transactions cause significant financial losses and security risks.The objective of this project is to:
Detect fraudulent transactions in real time, Reduce financial risk

# Dataset Information
Dataset contains transaction records Includes anonymized features
Target column indicates:
0 → Not Fraud
1 → Fraud
The dataset is preprocessed before training.

# Application Workflow

User enters transaction details.
Input data is converted into model-compatible format, Model predicts fraud probability, Threshold is applied.
